# Shecret

## Description
SSH and SFTP Client

![](../../sean.jpg)

## Stack
* Rust
* SQLite

## Features:

* Start SSH connection
* Start SFTP connection
* Create SSH keys
* List connections
* Interactive user-friendly CLI
* Standalone executable project. Just run it without installing any dependencies
