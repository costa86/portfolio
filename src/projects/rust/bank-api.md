# BankingAPI

## Description
Blazing fast REST API to handle basic banking operations

## Stack
* Rust
* Actix-web
* SQLite

## Features:

* Create customers
* Deposits
* Withdrawals
* Transfer money among customers
* Payments
* JWT authentication
* API Documentation available [here](https://documenter.getpostman.com/view/22809238/2s93CPqCBi)
