# ChatGPT voice assistent

## Description

Prompts the 2023 popular AI prompt ChatGPT via voice, instead of text

## Stack

* Python
 
## Features:

* Prompts ChatGPT with microphone input
* Saves response as audio file
* Saves response as text file
