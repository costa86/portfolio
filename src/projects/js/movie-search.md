# Movie Search

## Description
Web app to search and display information about movies, TV shows and games. The data is obtained from an API

## Stack
* HTML
* CSS
* SASS
* React
* JavaScript

## Features:

* Search for a movie/serie.
* See list of results.
* See each movie's details page.
